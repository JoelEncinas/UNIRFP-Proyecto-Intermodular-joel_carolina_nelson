# UNIRFP-Proyecto-Intermodular-joel_carolina_nelson-
Desarrollar una aplicación web completa que permita la gestión integral de un sistema de bicicletas urbanas compartidas, facilitando la reserva, consulta y uso de bicicletas por parte de los usuarios, y promoviendo un modelo de movilidad urbana sostenible apoyado en herramientas digitales.
